 Credit Card Transaction & Customer Report

 Overview

 This project is a Power BI Dashboard that analyzes Credit Card Transaction and Credit Card Customer data to generate meaningful business insights through interactive visualizations.

Tools & Technologies

- Power BI
- DAX (Data Analysis Expressions)
- Microsoft Excel
- SQL (for data support and analysis)

 Key Insights

- Credit Card Transaction Analysis
- Customer Report & Segmentation
- Revenue and Transaction Trends
- Interactive KPIs and Visualizations

 Project Objective

The goal of this project is to transform raw customer and transaction data into interactive dashboards that help in better business decision-making.
